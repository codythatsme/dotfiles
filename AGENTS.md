# DOTFILES

macOS dev env. zsh + Zed + Claude Code + bun + fnm. Stow-based.

## STRUCTURE

```
.
├── Brewfile              # `brew bundle` source of truth
├── install.sh            # brew bundle + stow + sync skills
├── home/                 # Stowed to ~ via `stow -t ~ home`
│   ├── .zshrc
│   ├── .gitconfig
│   ├── .ssh/config
│   ├── .claude/{CLAUDE.md,settings.json,skills/}
│   └── .config/{gh,git,lazygit,ripgrep,zed}/
├── scripts/
│   ├── mattpocock-skills.txt          # allowlist for skill sync
│   └── sync-mattpocock-skills.sh      # symlinks chosen skills into home/.claude/skills
└── external/             # gitignored, vendored repos (mattpocock-skills clone)
```

## WHERE TO LOOK

| Task | Location |
|------|----------|
| Add Homebrew package | `Brewfile` |
| Shell config | `home/.zshrc` |
| Git config / aliases | `home/.gitconfig` |
| Global Claude prompt | `home/.claude/CLAUDE.md` |
| Claude Code settings | `home/.claude/settings.json` |
| Add a mattpocock skill | `scripts/mattpocock-skills.txt` (then re-run install.sh) |
| Add a custom Claude skill | `home/.claude/skills/<name>/SKILL.md` |
| Zed editor settings | `home/.config/zed/settings.json` |
| Zed keymap | `home/.config/zed/keymap.json` |
| lazygit config | `home/.config/lazygit/config.yml` |
| ripgrep defaults | `home/.config/ripgrep/config` |
| SSH host entries | `home/.ssh/config` |
| Symlink dotfile to ~ | drop into matching path under `home/`, run `install.sh` |

## CONVENTIONS

- `home/` mirrors `~`. Anything you put under `home/.foo` ends up at `~/.foo` after stow.
- Stow folds when target dir exists (e.g. `~/.config`), creating per-child symlinks.
- `install.sh` is idempotent. Re-run after pulling.
- Script config (allowlists etc.) lives in `scripts/`, NOT `home/` — it's not user config.
- Custom Claude skills are real dirs in `home/.claude/skills/`. mattpocock skills are symlinks managed by sync script.

## ANTI-PATTERNS

- Editing `~/.zshrc`, `~/.gitconfig`, etc. directly — change `home/...` instead.
- Hardcoding `/Users/codythatsme/...` — use `$HOME` / `$DOTFILES`.
- Committing `external/` — gitignored, regenerated by sync script.
- Committing files matching `*secret*`, `*token*`, `*credentials*`, `*.local` — gitignored.
- Adding `mattpocock-skills.txt` entries without running `install.sh` to refresh symlinks.

## COMMANDS

```bash
./install.sh                          # full bootstrap / refresh
stow --target="$HOME" --restow home   # only re-stow (no brew/skills)
./scripts/sync-mattpocock-skills.sh   # only refresh skill symlinks
brew bundle dump --file=Brewfile --describe --force  # snapshot current brew state
```
